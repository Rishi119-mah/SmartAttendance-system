# Smart Attendance System

An AI-powered Smart Attendance System built with Python, Flask, and OpenCV. This system uses Local Binary Patterns Histograms (LBPH) to recognize faces through the browser webcam and logs attendance into an SQLite database with predictive machine learning analytics.

## Prerequisites
To run this project on a Mac, you will need:
- Python 3.9 or higher installed
- A working webcam

## Step-by-Step Setup Instructions

### 1. Clone the Repository
First, open the Terminal on your Mac and download the code from GitHub:
```bash
git clone https://github.com/Rishi119-mah/SmartAttendance-system.git
cd SmartAttendance-system
```

### 2. Create a Virtual Environment
It is highly recommended to create a virtual environment so the project dependencies don't interfere with your Mac's global Python installation. Run:
```bash
python3 -m venv .venv
source .venv/bin/activate
```
*(You will know it worked if you see `(.venv)` at the beginning of your terminal prompt).*

### 3. Install Dependencies
Next, install all the required Machine Learning and Web server libraries:
```bash
pip install -r requirements.txt
```

### 4. Run the Application
Once everything is installed, start the Flask server:
```bash
python3 app.py
```

### 5. Access the Web App
Open your web browser (Safari or Chrome) and go to:
**http://127.0.0.1:5000**

## How to Use the System
1. **Register**: Go to the "Register" tab. Type your name, allow webcam permissions, and click Start Registration. Look at the camera until the model finishes training.
2. **Mark Attendance**: Go to the "Attendance" tab and click "Start Camera". When your face is recognized with high confidence, your attendance is automatically logged in the database.
3. **Analytics**: Go to the "Dashboard" to view predictive attendance analytics generated by the Scikit-Learn engine.
